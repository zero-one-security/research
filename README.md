# Zero One Security — Open Research

Open data and methods behind the figures Zero One Security publishes.

We are a security company, and our discipline is verifiable evidence, so we hold
our own public claims to the same standard. Every figure we cite on our site or
in our magazine is backed here: the inputs, their primary sources, and a method
you can run and check yourself.

Where no clean primary source exists yet, we publish a transparent estimate and
tighten it as better data arrives. Estimates carry version history, so the git
log is the audit trail.

## Studies

- [`norwegian-cyber-insurance/`](norwegian-cyber-insurance/) — how many Norwegian
  businesses carry cyber insurance. Working estimate: ~38% of businesses with
  employees (2024), triangulated from public primary data.

## How to cite

Use the repository citation (on GitHub, "Cite this repository", generated from
[`CITATION.cff`](CITATION.cff)), or cite a study directly with the "How to cite"
section in its README.
